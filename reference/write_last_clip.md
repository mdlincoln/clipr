# Write contents of the last R expression to the clipboard

Write contents of the last R expression to the clipboard

## Usage

``` r
write_last_clip(...)
```

## Arguments

- ...:

  Pass other options to
  [`write_clip()`](http://matthewlincoln.net/clipr/reference/write_clip.md).

## Note

This is a wrapper function for `write_clip(.Last.value)`
